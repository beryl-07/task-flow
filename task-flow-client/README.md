# Task Flow Client

## Tech Stack

- React.js
- TailwindCSS
- shadcn/ui
- tanstack/react-table
- Zod

## Running locally

1. Install dependencies using npm

   ```bash
   npm install
   ```

2. Copy the `.env.example` to `.env` and update the variables.

   ```bash
   cp .env.example .env
   ```

3. Start the development server

   ```bash
   npm run dev
   ```
